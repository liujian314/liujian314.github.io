毕业两个月以来的感受

发表于 2018-10-01 | 分类于 杂感
第一段写于2018-09-16
自离校以来，一直没有写东西，趁”山竹”路过，写点什么。



不知不觉两个月就过去了，时间真的过得很快。感受层面上的东西抽象复杂，先用一句话来概括毕业后的感受：
我觉得在校时，就像是人在很高的云桥上骑自行车，周围的景色都在桥下；而毕业后，便觉得人换成在立交桥上走了，周围都是高楼大厦，夹杂着车流。

夹杂的信息很多，我看看什么时候可以补上两张画，先文字阐释一下，做个理解备忘。

  1. 学生时可以看到的东西很多，视野可以很广，但是看不清，细碎；到了工作后，我就可以看到具体的东西了，但是视野也被具体的事物所遮挡，要自行拓宽。
  2. 学生时阳光可以每时每刻沐浴全身，工作时不然；
  3. 色彩不一样了，学生时是浅色系。而工作后是低饱和的各种色的混合，也就是高级灰；
  4. 一个是在描绘梦想，一个是深入现实。

这两个月其实过渡得很自然，几乎没有体会到那种网络文章上所说的毕业生刚出来社会遭遇的各种艰辛（如果把在公司通宵以及每天深夜回家这事不算的话……）。当然这话可能说得早了，不过这段时间就感觉上而言，体会并不强烈。
最大的体验就是掉发明显，没准还瘦了哈哈哈哈哈（其实我太瘦了，正巴不得变胖一点）。人人都说在猪厂是会变胖的，食堂排队时一看，那确实是大多数人都看着觉得肉肉的，可为啥我吃不到一个月就腻了，然后就在想，这不胖三斤就算了，可别缩水了三斤。或许是我太挑食了，不爱吃甜食的我瞬间少了一半的选择。。。

掉发有没增多不知道，在学校时也这样。不过我猜想定是不健康的作息导致的。这周基本上都是两点睡的，虽然可以八点半才爬起来，但终究是太晚睡了。然后我又特别喜欢回家后喝牛奶，然后吃点水果，试想想十二点后还在吃吃吃。。。

2018-10-01增加内容

新人培训总算是过去了，我也算是顺利地进了相对喜欢的设计组，松了一口气。这尾声里夹杂着各种滋味。“有些事情不是一定就得按着标准来，沟通应该放在第一位。”这是我的第一个比较深的感触。说白了就是好的沟通可以带来更多的可能性，适用性表现在方方面面，并非仅限于工作，生活也是。对于一起奋斗了三个月最后无奈退出的小伙伴，我觉得很可惜，因为这种事情本身是可以有其它可能的，能力是一方面，交流也是一方面。当然，我不是在否定他人的能力，而是我认为沟通是一个在能力之上的软实力，越是人情社会，就越是如此。

最后，我们就这样到各自新的组里搬砖了。昨天贴好标签，做好搬离工位的准备后已经是晚上了，三个人在空荡荡的办公桌旁一起打完游戏再回家，很难得，大家都心知肚明，以后或许就鲜有机会一起面对面打游戏了，每天一定会相互叫上一起到十楼吃饭这事就更不可能了。
这天晚上我所感受到的是学生时代的那种纯粹。事实上我觉得相对以往而言，现在刚出来的毕业生没有太多关于同事之间的芥蒂，即使已经毕业了，大家彼此依旧是以同学角度相认的，这个时期的同事之间是可以收获很深的情谊的，只可惜，往往后天的竞争把人的纯粹给磨去了。

上班之后能做自己喜欢的事情的机会少了很多，我觉得可能是自己时间规划有问题，这也直接导致我借来的书到现在都还没看完…

这次先写到这了。
