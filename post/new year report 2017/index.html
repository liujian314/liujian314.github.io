辞去旧年-个人总结

发表于 2018-01-05 | 分类于 杂感


元旦前朋友圈被各种18岁照片刷爆，我有点恍惚，“这个小孩是90后的”这句话到现在终于成了个假命题。不过，除了些许感慨之外，我并没有太多感触。早过了18，那年特别傻，若是细想来倒有些痛恨那年的自己。罢，往事提多了也显空乏。

今年圣诞刚过，元旦紧跟其后，无论是西方节日还是国内传统节日，自己作为年轻的一代，最能感受到节日氛围。但是没什么好开心的，与平常一样，不过是朋友圈多了些照片和文章。当然，这可能只是标准single 🐕的感受……哈哈

2017年对我来说是极其重要的，乃至改变人生轨迹的一年。因为今年大四、今年完成人生第一次实习、今年我决定探索其他行业、今年终于把两年的压力锅给甩了、今年开始验证少年地中海的传说…… 太多太多，我甚至不清楚自己是怎么熬过来的。


这篇是年终个人总结呀，不应该是严肃点的吗？开篇一堆废话，连个小标题都没有！？
对，反正没人看，当日记写。
一年很长，但是总结起来也就几个字——该有的有，没有的还是没有。 


关于校园

事实上我觉得真正能强烈感受到毕业压力的是在大三的时候，也许周边的人基本是师兄师姐，他们的陆续离开和相继就职让我直接接触毕业浪潮。那时的我看到华农的一草一木都会想多看几眼，想着以后回母校喝口华农自产鲜牛奶的感觉都不一样，分分钟就开始发呆。

不过发呆是从高中时就养成的爱好，不是从2017年开始修炼的。发呆归发呆，我也开始思考在学校这段忙碌日子的意义。那么多课程和附带的作业，我可以用几成认真的态度去完成，那么多学校里面的杂事，我可以用什么样的姿态去处理。


关于人际

今年越发觉得自己的人际泛而空，这里并不从功利角度去衡量，单纯从整个过程中熟悉度的变化来评断。微信联系人越来越多，有多少我会真正愿意去保持联系？我多次扪心自问。不，应该有联系也不是主动去联系的吧。人际非玄学，但若想把它当成一件事真正处理得十全十美，这就不是真的人际来往，变质了，最起码自己心态上没真正想有一个有意义的人际关系。我在人前可以落落大方，侃侃而谈，在熟悉的人前秒变大话里的唐僧，但我清楚自己终究是个内向的人，寻找安心之所，静静呆着比什么都来得更加舒心。

想起L曾分析过我的话，她说得很对，我是个被动的人。我想这不止是在情感上应验了这句话，在其它人际来往中也同样适用。

不知有多少人与我一样，不怕生，但是被动得像是怕生的小孩。
今年与好些人在短时间内熟络起来，但过不了一两个月，就重新被打上咒印，最后无论大小事，不再问津。

所以，到了年关，我算是明白了一些事，根本不是什么人际空泛，也无关内向外向，其实是自己懒，冷漠，孤独。


关于工作

说今年非常有意义，很大程度上是因为工作方面计划的进展。我在大二时就开始思考该怎么合理安排实习与求职的时间和计划。大三暑假前开始找实习，如愿地在自己所计划的时间完成了第一次实习，并收获了很多工作体验。这可以说是相当令人开心的事。虽然现在已经不再回头，也不打算继续往服装行业前进了，但老东家“CM”作为第一个实习的公司，真的给了我相当好的印象，我始终对它保持极高的尊敬，祝愿这个集团能够继续前进。

也是因为这番实习，我终于明白了自己更应该走什么路，什么工作更适合我的困扰在实习回来后逐渐被解开。果然还是喜欢更有创造力的事情啊，还是继续画画吧。

想起大一开始就两手抓的事（一边发展爱好，一边扎实专业），蛮有意义的。不然，即便想改行，也不见得就能顺利如愿。没变动的话，未来几年应该是做设计了，我希望自己的心态不会变。前几年是拼命学习提升自己的阶段，我得牢记着一定要学习、学习、学习、学习！别想着赚钱。

我明白一个真理，真的想做到有所成，就不要以物质作为目标。警以为戒。


关于情感

没有的还是没有，一半在说这事。对自己的评价中，我在这方面就是个小孩，真的有太多把握不住的事情，一直以来都这样，成长得很慢，比其他人慢。但愿能够随时间继续有所成长。
我强迫自己懂得，有些改变不了的就不去纠结了，去把加倍的努力放在能争取的方面吧。我有足够的自信，方方面面。

开个玩笑：多学学马X、郭X、王XX……

哈哈，可能是自己的问题，可能是自己依旧在幻想，忘不掉，也可能是自己太理想主义，苛求太多。但不能将就啊，我可不愿成为将就的大叔！虽然我基本对相貌不苛求，但是品性和智慧必不可少。不过话说回来，三观和品性还是得直接接触和交流才看得出来，我这个从不主动的人在这坎上算是废了，无长处还不主动，哈哈哈哈。
最终还是看缘分？都2018了，醒醒

这是互联网啊大哥，还是保守些好，说太多就不行了，万一真有人看呢？对吧。


这篇年终总结没有新年展望，我不太喜欢把未来计划详细说出来，这会让我奋斗动力减半的，藏心里慢慢实现不更美滋滋？
