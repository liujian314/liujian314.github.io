今日杂绪

发表于 2018-01-24 | 分类于 杂感

两个月以来从未这么早起过的我，今天竟然破天荒地在八点钟左右自然醒来了，醒来发现L正在发消息过来，感觉不可思议，似乎醒来就是为了能及时回复般，很巧。很久很久没长聊过了，大概有一年，还是两年？

如果回去有时间见上一面就好了。

关于早上的问题，我细想了想，“自信+努力”虽然不能从根本上避免那些我或者说是对方所忧虑的情况，但是作为基础，一定必不可少不是么？

说说其它的，今天早上另一件令我开心的事情是我发现自己记错双周报提交日期了！！！本以为是后天就交，没想到是看了下安排表上是下星期才交，吓到我了，差点以为又要“粗制滥造+熬夜赶工”了。我这个顶级拖延症到今天还在琢磨着设计思路啊…..没有好的设计思路让我止步不前。

关于回家的事，毕设的第一套成衣才刚见到个型，很多工作还没完成，然而到29号就要回去了，不知是否来得及。我想晚点再回了，现在正在拼命抢下个月7、8号的票，上天快祝福我能顺利在29号前抢到啊。

辞去旧年-个人总结

发表于 2018-01-05 | 分类于 杂感


元旦前朋友圈被各种18岁照片刷爆，我有点恍惚，“这个小孩是90后的”这句话到现在终于成了个假命题。不过，除了些许感慨之外，我并没有太多感触。早过了18，那年特别傻，若是细想来倒有些痛恨那年的自己。罢，往事提多了也显空乏。

今年圣诞刚过，元旦紧跟其后，无论是西方节日还是国内传统节日，自己作为年轻的一代，最能感受到节日氛围。但是没什么好开心的，与平常一样，不过是朋友圈多了些照片和文章。当然，这可能只是标准single 🐕的感受……哈哈

2017年对我来说是极其重要的，乃至改变人生轨迹的一年。因为今年大四、今年完成人生第一次实习、今年我决定探索其他行业、今年终于把两年的压力锅给甩了、今年开始验证少年地中海的传说…… 太多太多，我甚至不清楚自己是怎么熬过来的。

绘画&设计类——个人书签分享

发表于 2017-11-11 | 分类于 分享

偶尔会有人跟我聊起画画的技巧，怎样才可以系统性地提高自己的审美水平。我是个萌新，诸如此类的问题，实在不敢妄加指点。不过，我可以将自己平常所频繁用到的学习渠道与大家做个分享。

以下的网址都是从个人chrome书签上精心选取的，也是我个人认为比较实用的，有兴趣的伙伴可以探索并收藏，毕竟是自己书签上的常客，不胡乱推荐。

学习绘画提升视觉审美没有什么好捷径，多看点东西（不限于绘画和设计），看后多练就行。走出家门到处看的条件不是每个人都可以有，那就靠伟大的互联网了。
